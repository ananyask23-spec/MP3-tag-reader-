# MP3-tag-reader-
An MP3 Tag Reader is a desktop-based software that reads and displays ID3 tag information from MP3 files using the command line. It extracts details such as song title, artist, album, year, genre, and comments. The project can be extended into a tag editor, allowing users to modify and update MP3 metadata easily.

Technologies used 
 Function pointers  
 String operations 
 File I/O handling
